## Inciando o projeto

Primeiro instale os pacotes:

```bash
yarn
# or
npm install
```


```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) de seu navegador para ver a aplicação rodando.

## Lint

Este repositório tem um linter configurado, para ver se está tudo organizado:

```bash
yarn lint
# or
npm run lint
```
